# UNESCO Country Website — Sitemap

## URL Structure

```
/                           → Homepage
/about                      → About UNESCO [Country]
/about/history              → Office History
/about/team                 → Team Members
/about/partners             → Partners

/programmes                 → All Programmes
/programmes/{slug}          → Programme Detail

/news                       → News & Stories Listing
/news/{slug}                → Article Detail

/events                     → Events Listing
/events/{slug}              → Event Detail

/publications               → Publications Listing
/publications/{slug}        → Publication Detail

/countries                  → Country Profiles
/countries/{code}           → Country Detail

/designations               → UNESCO Designations
/designations/world-heritage → World Heritage Sites
/designations/biosphere     → Biosphere Reserves
/designations/creative-cities → Creative Cities
/designations/intangible    → Intangible Cultural Heritage

/contact                    → Contact & Location
/search                     → Search Results

/{slug}                     → Custom/CMS Pages
```
